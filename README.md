# الباقيات الصالحات — مصحف وتفسير وأذكار

تطبيق إسلامي متكامل، **صدقة جارية على روح المرحوم شاكر محمود الجبوري**.
المصحف كاملاً بالتلاوة والتفسير وأدوات الحفظ، أوقات الصلاة حسب الموقع، أذكار المسلم، أدعية للميت، والسبحة — بلا إعلانات.

## طريقة النشر (مهمّة)
هذا المشروع يُنشر كملف واحد مكتفٍ بذاته يعمل في **كل المتصفحات** بما فيها متصفحات واتساب/تيليجرام المصغّرة.

**ارفع إلى GitHub ثم Netlify محتويات هذا المجلد (الجذر):**
- `index.html`  ← الملف الرئيسي (يحوي كل شيء: الأنماط والكود بداخله)
- `manifest.json` و`sw.js`  ← للتثبيت كتطبيق والعمل دون إنترنت
- مجلد `assets/`  ← اللوغو والأيقونات
- `netlify.toml` و`_headers` و`.nojekyll`  ← إعدادات النشر والأمان

> يكفي لعمل التطبيق رفع `index.html` وحده، لكن لتفعيل التثبيت والأيقونات ارفع بقية الملفات معه.

## تعديل التطبيق مستقبلاً (مع الحفاظ على الهندسة)
الكود منظّم بمعمارية معيارية داخل مجلد **`src/`** (تفاصيلها في `ARCHITECTURE.md`).
1. عدّل ما تريد داخل `src/` (كل قسم في ملفه: `src/js/features/…` والأنماط في `src/styles/…`).
2. شغّل أمر البناء لتوليد `index.html` من جديد:
   ```
   python3 src/build.py
   ```
3. ارفع `index.html` و`sw.js` المُحدَّثين (زد رقم النسخة يُدار تلقائياً في build).
4. ادفع التغييرات إلى GitHub؛ ينشرها Netlify تلقائياً.

بهذا تجمع بين **تنظيم الكود** (وحدات مفصولة في `src/`) و**التوافق الشامل** (ملف واحد كلاسيكي للنشر).

## المصادر (تعمل تلقائياً في المتصفح)
نص القرآن والتلاوة: jsDelivr و`cdn.islamic.network`. التفسير: `spa5k/tafsir_api`. أوقات الصلاة: Aladhan.
